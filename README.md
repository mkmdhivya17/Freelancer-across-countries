# Freelancer-across-countries
The Freelancer Dataset contains detailed information about freelancers, the projects they work on, their skills, client feedback, earnings, and platform activities. It is typically used for data analysis, data cleaning, visualization.
Dependencies
Before opening or using this Excel project, ensure the following prerequisites are met:
Operating System
Windows 10 or later
(or macOS 10.15+ if using Excel for Mac)
Software Requirements
Microsoft Excel 2016 or later
Recommended: Excel 2019 / Excel 2021 / Microsoft 365
Installing
Follow the steps below to download and set up the Excel project.
1. Download the Project File
You can download the Excel file from any of the provided locations:
Email attachment
Shared cloud folder (OneDrive, Google Drive, Dropbox)
USB drive or local machine folder
Project submission portal (if applicable)
Make sure to save the file to a location where you have full read/write access.
2. Extract the Files (if zipped)
If the project is provided in a ZIP or RAR archive:
Right-click the file
Select Extract All
Choose a destination folder
Click Extract
Ensure all folders remain in the same structure after extraction.
3. Open the Excel File
Double-click the .xlsx or .xlsm file
If prompted, click Enable Editing
If your project contains macros, click Enable Content
Executing program
1. Open the Project Folder
2. Launch the Excel File
3. Enable Editing
4. Load Data
5. Interact With Dashboards/Sheets
HELP
1. Excel Shows “Read-Only”
Cause: File is opened from email, cloud, or protected folder.
Fix:Save the file locally using File → Save As
Ensure the destination folder allows write access
2. “External Data Source Not Found” Error
Cause: The project uses linked files (CSV/Excel) and the file path changed.
Fix:Go to Data → Queries & Connections
Edit the query → Change Source to your local file path
